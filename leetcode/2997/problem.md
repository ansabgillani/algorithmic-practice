# Minimum Number of Operations to Make Array XOR Equal to K

**Problem ID:** 2997

**Difficulty:** Medium

## Description
Given a `0-indexed` integer array `nums` and a positive integer `k`, return the minimum number of operations required to make the bitwise XOR of all elements of the final array equal to `k`.

### Example 1:
**Input:** `nums = [2,1,3,4]`, `k = 1`
**Output:** `2`

### Example 2:
**Input:** `nums = [2,0,2,0]`, `k = 0`
**Output:** `0`

## Constraints
- `1 <= nums.length <= 10^5`
- `0 <= nums[i] <= 10^6`
- `0 <= k <= 10^6`
